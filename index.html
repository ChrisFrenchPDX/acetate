Acetate

A Node.js Static Site Generator

Extensiblity

Bring in custom data with Node or write an extenstion to manipulate the extend Acetae's build process.

Integration

Integrate Acetate into your existing toolchain and build process, whether its Gulp, Grunt or just the command line.

Focus

Acetate only handles templating and building pages and lets you use whatever you want to build the rest of your application.

Quick Start

To install Acetate, install the command line runner globally:

  $ npm install -g acetate-cli

Then you can clone the Acetate sample project and cd into it:

  $ git clone https://github.com/patrickarlt/acetate-sample.git && cd ./acetate-sample

Intall the dependencies with NPM:

  $ npm install

Run the built-in development server:

  $ acetate server --open
